# 🚗 US Accidents EDA (500K Sample)

This project explores traffic accident data from the US using a 500,000-row sample to reveal patterns by time, location, weather, and severity.

📊 Dataset: [Kaggle – US Accidents (March 2023)](https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents)
